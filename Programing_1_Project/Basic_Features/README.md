# Basic Features outlined on the Project requirement

#### Function 1:
* Add an Expense: Store the date, category, amount, and description in a file

#### Function 2:
* View All Expenses – Read and display all expense records.

#### Function 3:
* Search for Expenses – Filter records by date, category, or amount range.

#### Function 4:
* Update an Expense – Modify an existing entry.

#### Function 5:
* Delete an Expense – Remove a specific expense record.

#### Function 6:
* Error Handling – Ensure correct input formats and handle missing files.
